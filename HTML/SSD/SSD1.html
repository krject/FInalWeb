威剛 Ultimate SU800 固態硬碟採用最新的3D快閃記憶體，具備更高的存儲密度、效率與可靠度；並搭載智慧型 SLC Caching 與 DRAM Cache Buffer 技術，可大幅提升讀寫速度！另支援高 TBW ( 總寫入位元組數 )、DEVSLP ( 待機休眠省電 ) 與 LDPC ECC 除錯機制等功能，不僅耐用持久，也可確保資料傳輸的正確性。此外，使用者還可免費下載威剛獨家開發的加值軟體 「 SSD Toolbox 」 與 「 Migration Utility 」，不論是管理固態硬碟或移轉資料，都輕鬆省時。
Ultimate SU800 搭載 3D NAND Flash 技術，採立體堆疊的方式，比起傳統的 2D 平面 NAND Flash，可提供更高的儲存密度與容量，可靠度和耐用性也更上層樓！在 3D NAND Flash 已成主流的時代，Ultimate SU800 是兼具高效能與 CP 值的升級首選！
Ultimate SU800 共有 128GB 到 2TB 等完整的容量選擇，並藉由獨有的資料保護及修正技術，大幅提高其可承受的總寫入資料量 ( TBW, Total Bytes Written ) 及耐用度，滿足各種儲存應用的需求。
採用了業界最先進的 SLC 快取演算法技術，讓 Ultimate SU800 開啟快閃記憶體 SLC 模式，能大幅提升讀寫速度；並搭載 DRAM Cache Buffer，與不支援 DRAM 的 SSD 方案相比，隨機傳輸效能可提高逾兩倍以上！
Ultimate SU800 支援 LDPC ( Low Density Parity Check Code，低密度機奇偶檢查碼 ) 錯誤校正機制。在此項多層除錯機制下，可有效保護資料安全，並使快閃記憶體的耐用度及可靠度大幅提升！

Ultimate SU800 支援 DEVSLP ( 待機休眠省電 ) 功能，不僅降低消耗功率與電力成本，更能延長電池的使用壽命。在待機模式下，Ultimate SU800 的耗電量僅傳統硬碟的 1/200；在一般日常運作時，則比傳統硬碟省電 80%，可說是長時間行動辦公時的最佳夥伴。
相較於傳統硬碟，Ultimate SU800 可提供更優異的運算效能，不論是開機或讀寫檔案，都能節省更多時間，大幅提升工作效率！
